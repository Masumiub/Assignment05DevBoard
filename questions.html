Question-1: What are the different ways to select an element in the DOM?

There are different ways to select an element in the DOM. Such as:
1. getElementById(): This selects a single element by its unique id.
2. getElementsByClassName(): This selects all elements with a class name.
3. getElementsByTagName(): This selects all elements with a html tag.
4. querySelector(): This selects the first element matching a CSS selector.
5. querySelectorAll(): This selects all elements matching a CSS selector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Gets HTML content which includes tags and formatting inside a HTML element.
innerText: Gets visible text only. It does not show any hidden text. It doesn't include HTML tags.
textContent: Gets all text content, including hidden text. It does not parse HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where we add a event listener to a parent element, instead of adding event listener to each element.
For example, we have a unordered list id="parentList" .
< ul id = "parentList" >
< li class = "item" > Item 1 < / li >
< li class = "item" > Item 2 < / li >
< li class = "item" > Item 3 < / li >
< / ul>

document.getElementById('parentList').addEventListener('click', function (event){
event.target.parentNode.removeChild(event.target);
})
If we click on Item 1, Item 1 will be removed.

When we click a li, the click event bubbles up to the ul and then removes the targeted child.

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept where we click on a element or when a event occurs, that event bubbles up to its parent and ancestor elements in the DOM, until it reaches the root element. For any html file, document is the root element.


For example: We have a button in a card.
< div id = "card" >
< button id = "btn" > Click Here... < / button >
< / div >

document.getElementById('card').addEventListener('click', () => {
console.log('Card clicked');
}); document.getElementById('btn').addEventListener('click', () => {
console.log('Button clicked');
});

When we click the button:
Output will be:
Button clicked
Card clicked

Here, if we click on the button, the event bubbles up to the div(card) and shows output like, Button clicked, Card clicked.
Question-5: How do you create, add, and remove elements using JavaScript?

There are many ways to add or remove elements in js. Such as:
1. Create Elements:
let newp = document.createElement('p');
This creates a < p > paragraph. We can also add attributes, text, or innerHTML.

2. appendChild:
document.body.appendChild(newElement);
This newElement is Appended to it's Parent(Body) at the end.

3. Remove Elements:
let customBox = document.getElementById('customBox');
customBox.remove();
Here, we are removing a element, id named 'customBox', using remove() function.
