Question-1: What are the different ways to select an element in the DOM?

There are different ways to select an element in the DOM. Such as:
1. getElementById(): This selects a single element by its unique id.
2. getElementsByClassName(): This selects all elements with a class name.
3. getElementsByTagName(): This selects all elements with a html tag.
4. querySelector(): This selects the first element matching a CSS selector.
5. querySelectorAll(): This selects all elements matching a CSS selector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Gets HTML content which includes tags and formatting inside a HTML element.
innerText: Gets visible text only. It does not show any hidden text. It doesn't include HTML tags.
textContent: Gets all text content, including hidden text. It does not parse HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where instead of attaching event listeners to multiple child elements, we attach a single event listener to a parent element.
That single listener takes advantage of event bubbling, so it can "catch" events from child elements when they bubble up to the parent.
For example:
< ul id = "parentList" >
< li >Item 1< / li >
< li >Item 2< / li >
< li >Item 3< / li >
< / ul>

document.getElementById('parentList').addEventListener('click', (event) => {
if (event.target.tagName === 'LI') {
console.log('Item clicked:', event.target.innerText);
}
});

If we click on item 1, the output will be: "Item clicked:" "Item 1"

When we click a < li >, the click event bubbles up to the < ul >.
The listener on < ul > sees the event and checks if the event.target was a < li >.

Question-4: What is event bubbling in the DOM?

Event bubbling is a behavior in the DOM where an event triggered on a child element "bubbles up" to its parent elements.
When we click, input, or submit on an element, the event first fires on the target element (where the action happened), and then it bubbles up through all its ancestors (parents, grandparents, etc.) until it reaches the document object.


For example:
< div id = "parent" >
< button id = "child" > Click me < / button >
< / div >

document.getElementById('parent').addEventListener('click', () => {
console.log('Parent clicked');
}); document.getElementById('child').addEventListener('click', () => {
console.log('Button clicked');
});

When we click the button:
Output will be:
Button clicked
Parent clicked

Here, the click event fires on the button (#child).
Then the event bubbles up to the div (#parent), so the parentâ€™s listener also fires.
Question-5: How do you create, add, and remove elements using JavaScript?

There are many ways to add or remove elements in js. Such as:
1. Create Elements:
let newElement = document.createElement('div');
This creates a < div > element. We can also add attributes, text, or innerHTML.

2. appendChild:
document.body.appendChild(newElement);
This newElement is Appended to a Parent at End.

3. Remove Elements:
let elementToRemove = document.getElementById('someElement');
elementToRemove.remove();
Here, we are removing a element by id using remove() function.
